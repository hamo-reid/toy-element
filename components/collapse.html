Collapse 折叠面板 ​

折叠面板，提供可折叠的分组内容。

基础用法 ​

<script setup>
import { ref } from 'vue';
const activeNames = ref(['1'])
</script>

<template>
  <hm-collapse v-model="activeNames">
    <hm-collapse-item title="标题1" name="1">
      内容1
    </hm-collapse-item>
    <hm-collapse-item title="标题2" name="2">
      内容2
    </hm-collapse-item>
    <hm-collapse-item title="标题3" name="3">
      内容3
    </hm-collapse-item>
  </hm-collapse>
</template>

禁用状态 ​

通过设置 disabled 属性，可以禁用某一项。

<script setup>
import { ref } from 'vue'

const activeNames = ref(['1'])
</script>

<template>
    <hm-collapse v-model="activeNames">
    <hm-collapse-item title="标题1" name="1">
      内容1
    </hm-collapse-item>
    <hm-collapse-item title="标题2" name="2">
      内容2
    </hm-collapse-item>
    <hm-collapse-item title="标题3" name="3" disabled>
      内容3
    </hm-collapse-item>
  </hm-collapse>
</template>

手风琴 ​

通过设置 accordion 属性，可以开启手风琴模式，每次只能展开一个面板。

<script setup>
import {ref} from 'vue'

const activeNames = ref(['1'])

</script>

<template>
<hm-collapse v-model="activeNames" accordion="true">
    <hm-collapse-item title="标题1" name="1">
      内容1
    </hm-collapse-item>
    <hm-collapse-item title="标题2" name="2">
      内容2
    </hm-collapse-item>
    <hm-collapse-item title="标题3" name="3">
      内容3
    </hm-collapse-item>
  </hm-collapse>
</template>

自定义标题 ​

通过 title 具名插槽可以自定义标题内容。

<script setup>
import { ref } from 'vue'
const activeNames = ref(['1'])
</script>

<template>
  <hm-collapse v-model="activeNames" >
    <hm-collapse-item name="1" >
      <template #title>
        <span style="color: coral;">标题1</span><hm-icon icon="star" />
      </template>
      内容1
    </hm-collapse-item name="2">
    <hm-collapse-item title="标题2">
      内容2
    </hm-collapse-item>
  </hm-collapse>
</template>
